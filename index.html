HTML

  • The head element contains information about the webpage.
  • Elements inside the head are not typically seen
  • HTML attributes are always defined in the beginning element, never the closing element.
  • A !DOCTYPE html declaration tells the browser what document type it should expect when it is opened.
  • Some of the head elements include:
    • title: defines the title of a webpage
    • style: defines the CSS style associated to webpage
    • base: defiens the base URL for a webpage,
    • link: connects an external resource to the HTML document
    • meta: defines the metadata such as character set, descriptions and keywords for the website
    • script: embeds client-side scripts in the HTML document
  • The body element represents the visible content shown to the user.
  • The body element is an example of a semantic HTML - the element name describes its content and the role it plays on the website
  • Semantic HTML is best practice to use as it makes it easier for other developers to read, but also makes the website more accessible for those who use assitive technologies such as screen readers
  • Some of the body elements include:
    • h1-h6: represents the level of the heading a given text block represents with 'h1' being the largest
    • p: represents a paragraph or text block
    • ul, ol, li: the elements represent unordered, ordered and list items respectively and create bullet or numbered lists
    • img: contains information about images displayed on the webpage and contain attributes within it such as src (the location of the image), the alt text and the width and height
    • br: creates a line of empty space or a line break between two content blocks

CSS

  • It is possible to style an element directly in the HTML file using inline CSS. This internal stylesheet contains CSS rules within the head section of a HTML file.
  • It is important to consider seperation of concerns however as each section of code should have its own responsiblity
  • To link the external style sheet to a html file, we use a link element. This element contains two attributes rel ( the relationship of the document with the linked source) and href (specifies location of the external resource)
  • A CSS rule consists of a selector and a declaration block
  • A period preceding a section class title designates it as a class selector (eg .card). Any html elements assigned to the class will gain property assignments from the CSS
  • Assigning colors in CSS can be done in many ways, including by semantic references such as gray or blue or by its hexadecimal code
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git add -A: stages and commits the all files within the branch
  • git commit -m “”: -m flag adds a commit message and commits files locally
  • git pull origin main: pulls any changes from the current branch to ensure everything is up to date
  • git push origin branch-name: pushes any changes to github
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a single variiable that is used to hold a group of data
  • An array index is the location of an item in an array
  • JavaScript arrays are zeroindexed and sequential - the first item has an index of 0, not 1 and will always incread by whole numbers each time
  • A function is a set of instructions that tells the computer how to perform a certain task.
